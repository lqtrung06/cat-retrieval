# Hound Docs

Hound is a side-project SaaS for searching large personal media libraries with natural-language retrieval.

## Document Index

- [Functional Requirements](functional-requirements.md): MVP features, user stories, acceptance criteria, and non-goals.
- [User Flows](user-flows.md): core flows for onboarding, upload, indexing, search, and preview.
- [Data And Privacy](data-and-privacy.md): media handling, retention, privacy expectations, and safety requirements.
- [MVP Roadmap](mvp-roadmap.md): phased build plan for a lightweight launch.
- [WISE Frontend Reference](wise-frontend-reference.md): observed WISE frontend behavior to preserve or adapt.
- [Self-Grilled Decisions](self-grilled-decisions.md): follow-up v1 choices made without further user prompts.

## Current Product Statement

Hound is a SaaS for users with large collections of pictures and videos to find media through semantic search powered by open-source multimodal retrieval models, replacing manual checking through individual files.
